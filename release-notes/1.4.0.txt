## Added
- Вкладка настроек «Оверлей»: прозрачность (0–100%), угол экрана и выбор монитора
- Вкладка настроек «Редактор»: постобработка распознанного текста нейросетью
  - Локальные провайдеры: Ollama, LM Studio (порт)
  - Облачные провайдеры: OpenAI, OpenRouter, Groq, DeepSeek и свой OpenAI-совместимый API (ключ/URL)
  - Настраиваемый промпт с плейсхолдером {text}
- Подтверждение скачивания модели при переключении, если её ещё нет
- Автообновление приложения через GitHub Releases (проверка при запуске и вручную)

## Changed
- Оверлей пропускает клики сквозь себя (click-through)
- Реальный прогресс скачивания моделей и распознавания речи
- Убрана дублирующая строка статуса процесса в шапке

## Fixed
